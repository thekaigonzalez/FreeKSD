Assert(0L) A Kux Programmer's Function manual Assert(0L)

SYNOPSIS

    require("libmemio")

    func libmemio:Assert(bool expression)

DESCRIPTION

    Evaluates an expression and if the expression returns true, It will continue running the Lua application,
    if not, It will print an error to stderr and exit the program. This is good for debugging
