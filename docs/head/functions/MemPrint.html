NAME

    MemPrint(2L) - Print Allocated String with bytes

SYNOPSIS

    require("libmemio")

    func libmemio:MemPrint(int memory, string str)

DESCRIPTION

    Prints an allocated string to stdout.
    Allocated strings can have any number of memory, therefore the memory
    is freed after it has been used.
