LIBFS - Library tailored for gathering POSIX directories

    Libfs is tailored for one thing and one thing only, Directories.
    Containing one function, getworkingdir_posix(), it gets the working directory and returns it.

INIT SYNOPSIS

    int luaopen_libfs(lua_State* L);

FUNCTION SYNOPSIS

    int cwdfs(lua_State* L);

SEE ALSO

    LIBSYS

EXAMPLE

    -- Include the main file
    require('libfs')
    local dir = getworkingdir_posix()

    print(dir) -- /Users/Kai/Tests/

REGISTRY

    getworkingdir_posix() - Gets the current working directory

BUGS

    Does not return FULL FILE PATH, getworkingdir_posix() only returns the directory in which the environment is running.
