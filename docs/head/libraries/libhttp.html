LIBHTTP - Contains HTTP TCP Requests using C++ libraries.

    LIBHTTP contains HTTP TCP Requests which use LibCURL and a wrapper for libCURL called CPR

INIT SYNOPSIS

    int luaopen_libhttp(lua_State *L);

FUNCTION SYNOPSIS

    int gRString(lua_State* L);

    int gDownload(lua_State* L);

REGISTRY

    hgetstring(string website) - Returns a string with the given GET request return value text.
    hdofile(string website, string file) - Creates a file with the given GET request return value text. Doesn't return anything.
