LIBSYS - Contains filesystem calls using STD APIs and system calls.

    LIBSYS contains a bunch of functions for managing system calls and directories using Kux APIs.

INIT SYNOPSIS

    int luaopen_libsyscalls(lua_State* L);

FUNCTION SYNOPSIS

    int clearsys_posix(lua_State* L);

    int clearsys_winapi(lua_State* L);

    int ls_return(lua_State* L);

    int mkdir_sys(lua_State* L);

    int invoke_command(lua_State *L);

    int copy_files(lua_State* L);

    int sys_writeregistry(lua_State* L);

REGISTRY

    clear_posix() - Clears the screen using POSIX APIs

    clear_windows() - Clears the screen using Windows APIs

    sys_listdir(string directory) - Lists a directory using STD APIs

    sys_mkdir(string directory) - Creates a directory using STD APIs

    invoke_cmd(string command) - Invokes a command using STD APIs

    sysfcpy(string __fdir1, string __tdir2) - Copies FILES OR DIRECTORIES to another using STD APIs.

    sys_register(string vname, string vval) - Registers a variable into usr/bootloader using STD APIs.

DESCRIPTION

    LIBSYSCALLS contains system calls that function in bootloaders and bootchooser.lua
